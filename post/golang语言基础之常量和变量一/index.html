常量声明

常量定义

使用关键字const定义，常量是恒定不变的值，多用于定义程序运行期间不会改变的那些值。

//在常量或者变量函数等命名时首字母大写，意味着包外可访问
const Pi = 3.1415926
const Ch = "中国"

注意：常量声明时，Go的常量定义可以限定常量类型，如果定义常量时没有指定类型是无类型常量。

const Pi  float64 = 3.1415926
//Ch看着声明的是字符串类型，但是属于无类型常量，但是和字符串类型兼容，无需要转换
const Ch  = "中国"

多个变量可以一起声明，常量定义的右值也可以是一个在编译期运算的常量表达式。

const (
	Pi = 3.1415926
	Ch = "中国"
  Num1 = 1 << 3   //左移运算符，相当于1乘以2的3次方等于8
  Num1 = 8 >> 3   //右移运算符，相当于8除以2的3次方等于1
)

//注意：
const Home = os.GetEnv("HOME") //常量不能是程序运行期间才能获取到值的
预定义常量

Go语言预定义了这些常量：true、false和iota。

iota比较特殊，可以被认为是一个可被编译器修改的常量，在每一个const关键字出现时被重置为0，然后在下一个const出现之前，每出现一次iota，其所代表的数字会自动增1。(iota可理解为const语句块中的行索引)。

//示例：
const (
		n1 = iota //0
		n2        //1
		n3        //2
		n4        //3
	)

//1.用下划线跳某个值
const (
		n1 = iota //0
		n2        //1
		_         //跳过2
		n4        //3
	)

//2.声明中间插队
const (
		n1 = iota //0
		n2 = 100  //100
		n3 = iota //2
		n4        //3
	)
	const n5 = iota //0

//3.位移运算
const (
		_  = iota             //跳过0
		KB = 1 << (10 * iota) // 1*2^10 = 1024
		MB = 1 << (10 * iota) // 1*2^20 = ...
		GB = 1 << (10 * iota) // ...
		TB = 1 << (10 * iota)
		PB = 1 << (10 * iota)
	)

//4.多个iota定义在一行
const (
		a, b = iota + 1, iota + 2 //1,2
		c, d                      //2,3
		e, f                      //3,4
	)

//5.常用于枚举定义
const ( 
 Sunday = iota
 Monday 
 Tuesday 
 Wednesday 
 Thursday 
 Friday 
 Saturday 
 numberOfDays // 小写开头这个常量包外访问不了
)

变量声明

程序运行过程中的数据都是保存在内存中，我们想要在代码中操作某个数据时就需要去内存上找到这个变量，但是如果我们直接在代码中通过内存地址去操作变量的话，代码的可读性会非常差而且还容易出错，所以我们就利用变量将这个数据的内存地址保存起来，以后直接通过这个变量就能找到内存上对应的数据了。

Go语言中的变量需要声明后才能使用，同一作用域内不支持重复声明。 并且Go语言的变量声明后必须使用。

标准声明

Go语言的变量声明格式为：

var  变量名  变量类型

// example：
var name string
var age int
var isOK bool
批量声明

每声明一个变量就需要写var关键字会比较繁琐，go语言中还支持批量变量声明：

var (
    a string 
    b int
    c bool
    d float32
)
变量初始化

Go语言在声明变量的时候，会自动对变量对应的内存区域进行初始化操作。每个变量会被初始化成其类型的默认值，例如： 整型和浮点型变量的默认值为0。 字符串变量的默认值为空字符串。 布尔型变量默认为false。 切片、函数、指针变量的默认为nil。

当然我们也可在声明变量的时候为其指定初始值。变量初始化的标准格式如下：

var 变量名 类型 = 表达式

举个例子：

var name string = "leon"
var age int = 18

或者一次初始化多个变量

var name, age = "leon", 20
类型推导

有时候我们会将变量的类型省略，这个时候编译器会根据等号右边的值来推导变量的类型完成初始化。

var name = "leon"
var age = 18
短变量声明

在函数内部，可以使用更简略的 := 方式声明并初始化变量。

package main

import (
	"fmt"
)
// 全局变量m
var m = 100

func main() {
	n := 10
	m := 200 // 此处声明局部变量m,会覆盖全局变量声明的初始值
	fmt.Println(m, n)
}
匿名变量

在使用多重赋值时，如果想要忽略某个值，可以使用匿名变量（anonymous variable）。 匿名变量用一个下划线_表示，例如：

func foo() (int, string) {
	return 10, "leon"
}
func main() {
	x, _ := foo() //忽略字符类型的第二参数值leon
	_, y := foo() //忽略整型的第一个参数值10
	fmt.Println("x=", x)
	fmt.Println("y=", y)
}

匿名变量不占用命名空间，不会分配内存，所以匿名变量之间不存在重复声明。

注意事项：

  1. 函数外的每个语句都必须以关键字开始（var、const、func等）
  2. :=不能使用在函数外。
  3. _多用于占位，表示忽略值。
